# Créer une nouvelle piste audio sur garageband
### 1. Lancer Garageband

<img width="1439" alt="Capture d’écran, le 2021-11-13 à 13 59 38" src="https://user-images.githubusercontent.com/93773484/141656911-d3454a5d-d222-4c6d-9d95-72a97265473e.png">

> Page d'ouverture de Garageband

### 2. Appuyer sur le plus afin de choisir le type de piste (Option + Commande + N)

<img width="1440" alt="Capture d’écran, le 2021-11-13 à 13 59 49" src="https://user-images.githubusercontent.com/93773484/141656960-4e7a199b-dc3f-46c1-8e6f-cab1b8c1a4b2.png">
Vous pouvez aussi aller dans Choisissez Piste > Nouvelle piste

Choisir entre l'instrument logiciel, le microphone, l'instrument branché ainsi que le batteur.

### 3. Choisir l'entrée

<img width="1439" alt="Capture d’écran, le 2021-11-13 à 14 00 09" src="https://user-images.githubusercontent.com/93773484/141657134-296d0d2f-3ca1-484c-8f2b-2cabd3ee4686.png">

### 4. Vérifier les périphériques d'entrée et de sortie
<img width="1440" alt="Capture d’écran, le 2021-11-13 à 14 00 19" src="https://user-images.githubusercontent.com/93773484/141657304-05bb14c1-abda-4de7-96b2-460b2d83c9f3.png">
<img width="1440" alt="Capture d’écran, le 2021-11-13 à 14 01 24" src="https://user-images.githubusercontent.com/93773484/141657309-b3194600-142c-496b-8987-323e39f256c2.png">

Appuyer sur *Mon instrument est branché à* ou *J'entend le son à partir de* afin d'avoir plus d'information

### 5. Quitter les préférences

Avant de valider, vérifier si vous voulez entendre votre instrument en appuyant sur le bouton *Je souhaite entendre mon instrument lorsque je joue et j'enregistre*

### 6. Appuyer sur le bouton créer
<img width="1439" alt="Capture d’écran, le 2021-11-13 à 14 58 42" src="https://user-images.githubusercontent.com/93773484/141657435-6024f908-5b57-4b3d-a156-8e2a69783d4d.png">
Voilà votre piste est créé et vous êtes prêt à enregistrer
